SPACE MEMORY GAME - HTML5/JavaScript Game
=========================================

INSTALLATION
------------
1. Simply upload all files to your web server
2. Open index.html in any modern browser

REQUIREMENTS
------------
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side requirements
- No database needed

FEATURES
--------
- 5 challenging levels
- 3D card flip animations
- Responsive design
- Score tracking
- Time-limited gameplay
- Level progression system

CUSTOMIZATION
-------------
1. To change card images:
   - Replace images in /assets folder
   - Update references in script.js

2. To modify game difficulty:
   - Edit levels array in script.js

3. To change colors:
   - Modify CSS variables in styles.css
